## ❔❔ Why React ❔❔
* 재사용은 생산적이다.
* React는 말했다. 지속적으로 변하는 데이터를 위한 앱 구축에 적합하다.

  !React  :  Dom조작 li사이에 child를 찾고 ... 등 등.
  
  React : 비즈니스 로직만 중요. 컴포넌트간의 관계, 데이터 흐름 처리, 모델과 아키텍쳐에 초점.
